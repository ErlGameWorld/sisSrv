# 玩家role_id规则 
    在64位系统下  小整数用60bit表示 1位表示符号位 剩余59位可以用 用正整数表示 其中用低15bit表示服务器id(0-32767)  中间12bit随机数(0-4095) 高32bit用于自增id
# 机器人Key规则 
      在64位系统下  小整数用60bit表示 1位表示符号位 剩余59位可以用 用负整数表示 
# 怪物Key规则           
      在64位系统下  小整数用60bit表示 1位表示符号位 剩余59位可以用 用负整数表示
